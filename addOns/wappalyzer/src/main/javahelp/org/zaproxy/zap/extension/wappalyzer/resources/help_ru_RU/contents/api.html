Technology Detection API

Через API доступны следующие представления:

Просмотры

  • listSites: Lists all the sites recognized by the Technology Detection add-on.
  • listAll: список всех сайтов и связанных с ними приложений (технологий).
  • listSite (site*): список всех приложений (технологий), связанных с указанным сайтом.
